## Framework: Streamlit
The frontend is a Python-based Streamlit application acting as the "Sovereign Node" for the user experience.

## Key UI Components

### 1. Sidebar (Session Controller)
- **Session Display:** Shows the current `session_id`.
- **New Chat Button:** Triggers a session reset and URL parameter update.
- **Debug Info:** Real-time visibility of the connected `API_URL`.

### 2. Chat UI
- **Input:** Standard Streamlit `chat_input`.
- **Display:** Chronological stream of messages with role-specific styling (User vs. Model).
- **Citations:** Appends a "Sources" footer to model messages when grounding metadata is available.

### 3. Session Management
- **Persistence:** The application checks for `session_id` in query parameters on startup.
- **Recovery:** If a parameter is found, it automatically fetches the history from the backend to restore context.
