Objetivo: Establecer los contratos de comunicación deterministas.




Alcance permitido: FastAPI con especificación OpenAPI 3.1.


Reglas técnicas obligatorias:


Idempotencia: Header X-Idempotency-Key obligatorio para POST /messages.



Errores Estándar: Usar RFC 7807 para todos los errores de API.



Paginación: Las sesiones previas en el sidebar MUST usar paginación por cursor.



Validaciones: Validar payloads con Zod/Pydantic; prohibido el uso de Any.
